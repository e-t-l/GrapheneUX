# Install TapTap

This one's easy. 
1. Install [TapTap](https://github.com/KieronQuinn/TapTap).
2. Follow the setup instructions.
3. (I do recommend enabling "Auto-restart service" in TapTap's settings, and toggling Exploit Compatibility mode in App Info, as I've noticed TapTap will randomly crash in the background sometimes.)
