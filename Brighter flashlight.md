# Bypass Flashlgiht HAL for maximum brightness

1. Install [PixelLight](https://github.com/chenxiaolong/PixelLight)
2. Follow instructions in PixelLight's Github repo
3. Optionally, assign it as a lockscreen shortcut in the role of "QR" or "Notes" (If assigned to QR, then the actual QR-Reader can't be accessed via lockscreen shortcut)
4. Optionally, assign it as a double-tap action in [TapTap](https://github.com/e-t-l/GrapheneUX/blob/main/Tap%20Phone%20Back.md)
